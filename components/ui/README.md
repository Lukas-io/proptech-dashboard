# Expert Investment - Property Investment Analytics Dashboard

A professional property investment analytics platform built with Next.js 16, TypeScript, and comprehensive SEO/GEO optimization for maximum discoverability.

## üöÄ Features

### Investment Analytics
- **Portfolio Dashboard** - Track portfolio performance, ROI, and cash flow with interactive charts
- **Property Management** - Manage 15+ investment properties with detailed financial metrics
- **Investor Relations** - Track investor relationships, investments, and portfolio performance
- **Opportunity Pipeline** - Manage due diligence, inspections, and market analysis requests
- **Proposal Tracking** - Review and approve investment proposals from partners
- **Task Management** - Organize investment tasks with priority levels and completion tracking

### SEO & Discoverability
- **Comprehensive Metadata** - Unique titles, descriptions, keywords for every page
- **OpenGraph & Twitter Cards** - Rich social sharing previews
- **JSON-LD Structured Data** - Organization, SoftwareApplication, FAQ, and HowTo schemas
- **Sitemap Generation** - Automated sitemap with custom priorities
- **Security Headers** - HSTS, CSP, X-Frame-Options, and more
- **Image Optimization** - AVIF and WebP format support
- **LLM-Friendly Content** - Semantic HTML and natural language optimization
